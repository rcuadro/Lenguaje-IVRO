Para probar la funcionalidad de nuestro compilador, se proporcionan una serie de ejemplos ilustrativos que muestran de lo que es capaz IVRO. Para ejecutar cada uno de ellos, basta escribir en la terminal el comando:

bash probar.sh n

Donde n es el número del ejemplo seleccionado. Los ejemplos disponibles en el directorio "/tests" son:

1 -> ejemplo_bucles.txt
2 -> ejemplo_ctes_condicionales.txt
3 -> ejemplo_denote_incase_matriz.txt
4 -> ejemplo_funciones_sobrecarga.txt
5 -> ejemplo_IO_punteros.txt
6 -> ejemplo_recursion.txt
7 -> ejemplo_tuplas.txt

**IMPORTANTE**
En el caso del ejemplo 5, introduce dos números. En el resto de casos, es necesario hacer Ctrl+C para terminar el programa.
